# Shadow Fox Internship - Task 2

This repository contains two intermediate-level Python projects created for the Shadow Fox Internship program.

## Projects

### 1. Web Scraper
Located in `Web-Scraper/` folder
- Extract data from websites using Beautiful Soup and Scrapy
- Support for multiple sites (quotes, news, books)
- Export to JSON and CSV formats
- Session management and error handling

### 2. Hangman Game  
Located in `Hangman-Game/` folder
- Word guessing game with visual hangman progression
- Hint system and custom word management
- Clean terminal interface with multiple game modes
- JSON-based word storage system

## Setup Instructions

### Clone Repository
```bash
git clone https://github.com/sach200/shadow-fox-internship.git
cd shadow-fox-internship
```

### Web Scraper Setup
```bash
cd Web-Scraper
pip install -r requirements.txt
python scraper.py
```

### Hangman Game Setup
```bash
cd Hangman-Game
python enhanced_hangman.py
```
