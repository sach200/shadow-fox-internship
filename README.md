# Python Beginner Tasks

This repository contains 5 Python programs covering fundamental programming concepts for beginners.

## Programs Included

### 1. Variables (`1_variables.py`)
- Creating variables and checking data types
- Understanding reserved keywords
- Simple Interest calculation

### 2. Numbers (`2_numbers.py`)
- Using format function with different representations
- Mathematical calculations (area, speed)
- Working with integers and floats

### 3. Lists (`3_lists.py`)
- List operations and manipulations
- Justice League superhero management system
- Adding, removing, and sorting list elements

### 4. If Conditions (`4_if_conditions.py`)
- BMI calculator with categories
- City to country mapping
- Conditional logic and comparisons

### 5. For Loops (`5_for_loops.py`)
- Die rolling simulation with statistics
- Interactive jumping jacks workout program
- Loop control and user input handling

## How to Run

1. Make sure you have Python installed on your system
2. Navigate to the project directory
3. Run any program using: `python filename.py`

Example:
```bash
python 1_variables.py
```

## Requirements
- Python 3.x
- No external libraries required (uses only built-in modules)

## Learning Objectives
- Understanding Python variables and data types
- Working with numbers and mathematical operations
- List manipulation and methods
- Conditional statements and logical operators
- Loop structures and user interaction
